### 2.3 数据集样例（3条）

**Q1**: "法国原子能和替代能源委员会在各个技术领域分别有多少专利？"
```cypher
MATCH (p:Patent)-[:APPLIED_BY]->(o:Organization) WHERE o.name CONTAINS '法国原子能和替代能源委员会'
MATCH (p)-[:BELONGS_TO]->(td:TechDomain) WITH td.name AS domain, count(DISTINCT p) AS count
RETURN domain, count ORDER BY count DESC
```

**Q2**: "有诉讼记录的专利总共有多少件？"
```cypher
MATCH (p:Patent) WHERE p.litigation_count > 0 RETURN count(DISTINCT p) AS total
```

**Q3**: "2015年以来，公司在氢燃料电池领域的专利申请数量呈现怎样的趋势？"
```cypher
MATCH (p:Patent)-[:APPLIED_BY]->(o:Organization {entity_type: '公司'}),
      (p)-[:BELONGS_TO]->(td:TechDomain {name: '氢燃料电池'})
WITH substring(p.application_date, 0, 4) AS year, count(DISTINCT p) AS count
WHERE toInteger(year) >= 2015 RETURN year, count ORDER BY year
```

## 三、实验结果（2026-03-11 评估）

### 3.1 Direct模式评估结果

| 模型 | 执行准确率(EX) | PSJS | 可执行率 | 语法错误率 | 平均延迟(ms) |
|------|--------------|------|---------|-----------|-------------|
| **GLM-5** | **61.40%** | **89.12%** | **100%** | **0%** | 24864 |
| DeepSeek V3.2 | 58.60% | 85.16% | 98.80% | 1.20% | 3314 |
| DeepSeek V3.1 | 56.80% | 85.36% | 99.00% | 1.00% | 2690 |
| Qwen3-235B | 55.00% | 85.45% | 99.80% | 0.20% | 2535 |
| ERNIE-4.5-Turbo | 54.40% | 83.12% | 98.40% | 1.60% | 4076 |
| Qwen3.5-35B | 51.00% | 78.77% | 95.20% | 3.20% | 2400 |

### 3.2 关键发现

1. **GLM-5准确率最高(61.4%)**: 可执行率100%、语法错误率0%，但延迟最高（~25s）
2. **DeepSeek系列性价比最佳**: 准确率仅次于GLM-5（56-58%），延迟低一个数量级（~3s）
3. **所有模型可执行率 > 95%**: 说明模型基本掌握了Cypher语法
4. **PSJS普遍在83-89%**: 即使执行结果不完全匹配，部分结果集重合度也较高
5. **Qwen3.5-35B（小模型）差距明显**: EX仅51%，语法错误率最高(3.2%)，但延迟最低
